## Notice

This crate has no dependencies on either the `embassy-rp` crate or any of the `esp-*` crates; these
dependencies are only added to workspace members where they have one chip-specific feature enabled.

## Minimum supported Rust version

The minimum supported Rust version for `mplusfonts-examples-common` is `1.89`.
